# Resturant Management System

### Description
Resturant Management System is an online Sweets and Bakery store that allows customers to browse through various Sweets & bakery products and place orders. The project is implemented using Spring Boot and provides RESTful APIs for managing products, categories, customers, carts, and orders. The application is designed to be user-friendly and secure, with different functionalities available for administrators and customers.


## Usage
### As an Admin, you can:
- Manage products and categories.
- View all users and customers.
- Manage orders and bills.

### As a Customer, you can:
- Browse products by category.
- Add products to your cart.
- Place and track orders.
- View your order history.

## Technology Stack
- Java
- Spring Boot
- Spring Data JPA
- MySQL (or any supported database by Spring Data JPA)
